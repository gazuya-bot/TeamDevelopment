売り上げ＆ポイント管理システム　　

## Overview. 
小売事業者様に向けた、顧客情報管理とともに、売り上げ＆ポイント管理ができるシステムとなります。  
売り上げデータをもとに、月間売り上げ・年度売り上げ管理をグラフで可視化できます。

ユーザー様会員登録画面

ホーム画面に、顧客名と保有ポイント数を表示. 
お買い上げ毎に、売り上げとポイント使用時のポイント数を入力して、保有ポイントが更新されます。  

## Requirement  
Laravel Framework 6.20.34.
